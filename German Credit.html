Для примера построим гистограммы распределения по возрасту. Как видно из графика, они почти совпадают. Для количественной оценки можно использовать PSI (Population Stability Index), он расчитывается по формуле

$$PSI = \sum{((n1_i/N1)-(n2_i/N2)) * ln((n1_i/N1)/(n2_i/N2))}$$

Здесь n1i/N1 - это частоты первого распредления, n2i/N2 - второго.

Если PSI < 0.1, популяция стабильна, если PSI > 0.2, сдвиг значительный, модель в существующем виде может дать плохие результаты.

Рассчитаем значение PSI для данного примера.

In [175]:
n1 = np.histogram(data1['age'])[0]
n2 = np.histogram(data2['age'])[0]

N1 = np.sum(n1)
N2 = np.sum(n2)
In [184]:
PSI = np.dot(np.log((n1*1.0/N1)/(n2*1.0/N2)), (n1*1.0/N1-n2*1.0/N2))
In [185]:
print PSI

0.0227876824861

В данном случае значение PSI меньше 0.1, так что выборка стабильна.

Для сравнения распредлений можно использовать критерий Хи-квадрат или тест на равенство средних без предположения о нормальности распределений (в данном случае из диаграмм видно, что оно не выполняется).

U-критерий Манна — Уитни (англ. Mann — Whitney U-test) — статистический критерий, используемый для оценки различий между двумя независимыми выборками по уровню какого-либо признака, измеренного количественно. Не требует нормальности распределений.

Используем его для проверки гипотезы о равенстве средних в выборках.

Мы можем провести такие тесты для всех численных переменных, с поправкой на множественную проверку гипотез, например, методом Холма. https://ru.wikipedia.org/wiki/Коррекция_на_множественное_тестирование Для категориальных придется использовать гистограммы по количеству, полученные из сводных таблиц pandas, и PSI.
